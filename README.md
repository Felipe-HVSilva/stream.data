# Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Felipe-HVSilva/stream.data
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
